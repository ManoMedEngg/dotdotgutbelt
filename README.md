# Gift Basket Birthday Web App

A small interactive web application based on a draggable gift box and a basket drop zone.

## Features
- Draggable gift box (mouse + touch)
- Drop into basket to trigger celebration
- Confetti, toast messages, modal, and Tamil thank-you message
- Reset button to replay
- Animated water droplet background

## Run locally
1. Open `index.html` in a browser (double-click or via web server).
2. Drag the gift box into the basket to unlock the animation and messages.

## Optional local server (recommended)
- Python 3:
  - `python3 -m http.server 8080`
  - Visit `http://localhost:8080`
